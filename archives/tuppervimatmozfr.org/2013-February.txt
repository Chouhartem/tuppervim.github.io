buffer qui se modifie pas comme je m'y attends et les capacités de
débuggage des scripts python qui s'approchent du néant...

Je propose donc pour un prochain bidouillonthons de s'attaquer à ce
plugin pour faire en sorte que cette nouvelle version puisse sortir et
en éradiquer le code perl ! Biensûr, sous couvert de coder sur ce
script, ce serait un bon workshop sur "comment coder un script python
